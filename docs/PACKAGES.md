# Standardized Service Packages

Define clear, repeatable packages to avoid custom chaos. Prices are examples — adjust for market.

1) Starter Website
- Pages: Home, About, Services, Contact
- Revisions: 1
- Turnaround: 7–10 days
- Hosting: Optional
- Maintenance: Optional (monthly)
- SEO: Basic on-page SEO (titles, meta)
- Support: Email support (48h)
- Price: $799 one-time

2) Business Website
- Pages: Home, About, Services (up to 5), Contact, Blog
- Revisions: 2
- Turnaround: 14–21 days
- Hosting: 1 year optional
- Maintenance: 3 months basic
- SEO: On-page + sitemap
- Support: Email + ticketing (48h)
- Price: $2,499 one-time

3) Premium Website
- Pages: Home, About, Services (unlimited), Contact, Blog, Case Studies, Team
- Revisions: 3
- Turnaround: 21–35 days
- Hosting: 1 year included
- Maintenance: 6 months included
- SEO: On-page + basic keyword research
- Support: Priority (24h)
- Price: $5,999 one-time

4) Maintenance Only
- Scope: content updates, backups, minor fixes (up to X hours/month)
- Revisions: ongoing within included hours
- Turnaround: SLA-based
- Hosting: separate
- Price: $99–$499/month depending on hours

5) Hosting + Maintenance
- Includes: hosting, SSL, daily backups, maintenance (Y hours/month)
- SLA: 99.9% uptime target
- Price: $49–$299/month depending on resources

Notes:
- Always include clear revision limits and out-of-scope items in proposals.
- Add optional add-ons: e-commerce, booking, integrations, custom dev (priced separately).
