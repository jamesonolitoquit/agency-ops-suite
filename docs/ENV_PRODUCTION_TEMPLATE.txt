# ============================================================================
# HOW TO SET THESE IN VERCEL
# ============================================================================
#
# 1. Go to your Vercel project dashboard
# 2. Click "Settings" → "Environment Variables"
# 3. For each variable above, click "Add"
# 4. Set Name = variable name
# 5. Set Value = the secret/value
# 6. Select environments: Production (not Preview or Development)
# 7. Click "Save"
#
# For secrets that rotate (like INTAKE_WEBHOOK_SECRET):
# 1. Copy the current secret
# 2. Set it in a new Preview variable first (for testing)
# 3. Update Production
# 4. Update your public site's matching variable
# 5. All sites update at next deployment
