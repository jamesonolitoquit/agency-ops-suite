# Admin Dashboard Docs Index

This folder follows a naming convention so files are easy to scan and sort.

## Naming Convention

- `GUIDE_*`: setup and how-to documentation
- `CHECKLIST_*`: step-by-step validation lists
- `REPORT_*`: progress and status summaries
- `REFERENCE_*`: technical references and API details
